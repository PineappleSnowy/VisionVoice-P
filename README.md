## 👋🏻 什么是视界之声VisionVoice

视界之声V4.0是一款帮助视障人士（以下简称为“您”）拍照和出行、寻物的无障碍软件，是您生活中的贴心助手。

通过人脸检测、语音识别和语音播报等高新技术，视界之声将辅助用户完成自拍；通过场景识别和环境检测等高新技术，视界之声将帮助用户分析附近的环境，从而辅助用户出行。

## 🏁 快速开始

以下操作如果对您来说不太方便，您可以通过您的家人、朋友来协助您进行操作。

### 1.安装

“视界之声”现已推出电脑体验版和手机正式版。

#### 电脑体验版安装

电脑体验版只有“智慧拍照”和“环境识别”功能可以体验。

免费[安装“视界之声”电脑体验版](https://visionvoice.life/%E5%BA%94%E7%94%A8/Camera.exe)

#### 手机正式版安装

手机正式版（即“视界之声”手机版）拥有 [在线演示](#-在线演示) 中展示的所有功能。

扫描二维码，免费安装“视界之声”手机版。

![download_app](readme_files/download_andriod_app.png)

### 2.免费注册并登录

1. 免费[注册账号](https://visionvoice.life)

2. 登录账号，在用户设置 > [账户与密码](https://pineapplesnowy.cn/settings) 里设置您的用户名和密码

您可以通过设置“一键登录”让“视界之声”信任您的设备，设置后在该设备打开“视界之声”时该账号将自动登录。您还可以设置声纹密码采用更方便的声纹登录。

#### 注意：免费版提供所有基础功能。如果您想拥有更人性化的智能体、更流畅的网络处理速度、更多的聊天次数、更准确迅速的识物寻物和更棒的体验，您可以购买“视界之声”的会员解锁一切高级功能。

登录账号，在用户设置 > [会员](https://pineapplesnowy.cn/settings/membership) 里选择您喜欢的套餐进行购买。
